--------------------
      qt-bdb
--------------------
"Integration project for Qt framework and Berkeley DB"
http://code.google.com/p/qt-bdb/

The goal of the qt-bdb project is to bring set of tools and libraries aimed
towards integrating Oracle Berkeley Database with Qt framework.

qt-bdb requires qmkbld package to build. Project home for qmkbld is
http://code.google.com/p/qmkbld/
